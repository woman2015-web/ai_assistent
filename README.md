# ИИ-ассистент Центра йоги «Прана»

## Описание проекта

Проект представляет собой интеллектуального ассистента на базе искусственного интеллекта, разработанного специально для Центра йоги «Прана». Цель решения — автоматизировать взаимодействие с клиентами, обеспечивая мгновенные и точные ответы на вопросы, а также упрощая процесс записи на онлайн-занятия. Ассистент доступен через популярный мессенджер Telegram, что делает его удобным и привычным для пользователей каналом коммуникации.

## Основной функционал

- **Осмысленное ведение диалога с помощью нейросети**  
  Ассистент способен поддерживать естественный, контекстно-зависимый диалог, понимая намерения пользователя и адаптируя ответы под текущую беседу.

- **Ответы на вопросы по базе знаний**  
  Вся информация о занятиях, расписании, преподавателях, методиках и правилах центра хранится в структурированной базе знаний. ИИ использует эти данные для предоставления точных и актуальных ответов.

- **Запись клиента на онлайн-занятие и добавление события в Google Календарь**  
  Пользователь может выбрать удобное занятие и записаться на него прямо в чате. После подтверждения система автоматически создаёт событие в календаре клиента через интеграцию с Google Calendar, отправляя напоминание и ссылку на подключение.

## Технологический стек

- **Qwen**  
  Использован для разработки системного промпта и формирования семантической базы знаний, обеспечивающей корректную и релевантную работу ассистента.

- **Савви (https://suvvy.ai/)**  
  Платформа для создания и развёртывания ИИ-ассистентов без необходимости глубокого программирования. На ней реализовано ядро ассистента, включая логику диалога, обработку запросов и интеграции.

- **Telegram API**  
  Обеспечивает двустороннюю связь между пользователем и ассистентом через официальный интерфейс Telegram, позволяя использовать все преимущества мессенджера: уведомления, кнопки, медиа и т.д.

## Текущий статус

Разработан и протестирован базовый функционал ИИ-ассистента. Успешно реализована интеграция с Telegram. Система готова к пилотному запуску и дальнейшему масштабированию (например, добавлению оплаты, персонализированных рекомендаций или аналитики).
